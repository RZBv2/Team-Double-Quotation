## ✨ Overview

**Online Exam System** is a cutting-edge, feature-rich examination management platform built with Python. Designed for educators and institutions that demand security, precision, and an exceptional user experience.

This system combines powerful anti-cheat mechanisms with an intuitive, modern UI to deliver a seamless testing environment that protects academic integrity while keeping test-takers engaged.

---

## 🔐 Key Features

### 🛡️ Advanced Security & Anti-Cheat
- **Real-time proctoring detection** — Monitors window switching and suspicious activity
- **Keyboard tracking** — Detects unauthorized inputs and shortcuts
- **Windows integration** — OS-level security checks (Windows optimized)
- **Session validation** — Cryptographic verification of test integrity

### 💼 Exam Management
- **Dynamic question banking** — Create, store, and manage unlimited questions
- **Flexible exam creation** — Customizable test formats and timing
- **Real-time grading** — Instant feedback and score calculation
- **Candidate tracking** — Monitor exam progress and performance metrics

### 🎨 Professional UI/UX
- **Modern gradient design** — Custom-styled widgets with cohesive color scheme
- **Responsive interface** — Built with Tkinter for cross-platform compatibility
- **Intuitive workflows** — Streamlined navigation for administrators and test-takers
- **Accessible design** — Clear typography and high-contrast visuals

### 💾 Data Management
- **SQLite database** — Secure local storage with encrypted credentials
- **Password hashing** — Industry-standard security protocols
- **Test logs** — Comprehensive audit trails and activity records
- **JSON configuration** — Easy customization of system parameters

## 📋 System Requirements

- **Python 3.7+**
- **Windows 10/11** (recommended for full anti-cheat features)
- **Dependencies:**
  - `tkinter` (included with Python)
  - `sqlite3` (included with Python)
  - `keyboard` (optional, for enhanced anti-cheat)
  - `pywin32` (optional, for Windows API integration)
